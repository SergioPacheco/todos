#### Usando o JAR executável

Para execução de um JAR:

```bash
$ mvn clean package
``` 

Para executar esse aplicativo, use o comando java -jar, da seguinte maneira:

```bash
$ java -jar target/todos-0.0.1-SNAPSHOT.jar
```
